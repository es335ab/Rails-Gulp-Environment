## Gulp Usage

### Installing packages and dependencies

- Global
```
npm install -g gulp
npm install -g easymock
```

- Local
```
npm install
bower install
```

- easymock
```
cd easymock
easymock -p 8080
```

### Gulp command

- development
```
gulp server
```

- deploy optim
```
gulp build
gulp optim
```
